For developers

Package your existing agent in 2 minutes. No code changes needed — just add a config file.

# Install the CLI
npm install -g @agentpak/cli

# In your existing agent repo, initialize
agentpak init --type project
✓ Created agentpak.toml

# Edit the manifest (name, entry point, permissions)
vim agentpak.toml

# Validate and pack
agentpak validate
✔ Valid agent configuration
agentpak pack
✔ Packed → my-agent-1.0.0.agent (12.3 KB)

# Add a badge to your README
agentpak badge --repo you/your-repo --file my-agent-1.0.0.agent
