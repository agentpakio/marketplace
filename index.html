For developers

Package your existing agent in under 5 minutes. No code changes needed.

Any framework

LangGraph, CrewAI, OpenAI SDK, or plain Python. If it runs, it packs.

Zero code changes

Add a config file and a thin bridge script. Your agent code stays untouched.

Instant distribution

One .agent file. Share via GitHub, Discord, email, or the marketplace.

Users love you

They double-click and it works. No Python install, no pip, no venv, no .env files.

# In your existing agent repo
agentpak init --type project
✓ Created agentpak.toml

# Configure name, entry point, permissions
vim agentpak.toml

# Validate and pack
agentpak validate
✔ Valid agent configuration
agentpak pack
✔ Packed → my-agent-1.0.0.agent (12.3 KB)

# Add a badge to your README
agentpak badge --repo you/your-repo --file my-agent-1.0.0.agent
